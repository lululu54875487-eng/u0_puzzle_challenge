# 社群介紹文

可以直接複製下面這段貼到社群。最後一行請換成你的 Render 遊戲網址。

```text
💜 小U0拼圖闖關開放挑戰啦！ 🧩✨

這是一個可以在社群一起玩的拼圖闖關小遊戲～
關主會先上傳一張圖片，設定拼圖難易度（低 2x2／中 3x3／高 4x4／魔王級 5x5）、限時或不限時，還會偷偷放入一組「破關暗號詞」🔐

🎮 玩法說明

1. 關主建立拼圖房
關主上傳圖片後，系統會自動把圖片變成拼圖並打散，還能選低、中、高、魔王級四種難易度。

2. 關主分享房號
建立完成後會得到一組房號，把房號貼到社群給大家挑戰。

3. 闖關者輸入房號
每個人進入後都會有自己的拼圖畫面，不會看到別人的進度。

4. 完成拼圖拿暗號
拼圖完成後，畫面會出現破關暗號詞。

5. 回社群喊出暗號
把暗號詞複製回社群留言，就算闖關成功！🎉

🌟 適合這樣玩
💬 社群互動
🎁 抽獎活動
🎂 生日驚喜
📸 圖片猜謎
🧠 小型闖關任務

準備好了嗎？
快輸入房號來挑戰小U0拼圖闖關吧！💜🧩

🔗 遊戲連結：
你的 Render 網址貼這裡
```
